# python-lekin

python-lekin is a **Flexible Job shop Scheduler Library**, named after and inspired by [Lekin](https://web-static.stern.nyu.edu/om/software/lekin/).


## Tutorial

**install**
```
pip install python-lekin
```

**usage**
```
import lekin

```
